Компьютерные языки разметки

Бра́узер, или веб-обозрева́тель — прикладное программное обеспечение для просмотра страниц, содержания веб-документов, компьютерных файлов и их каталогов.
Веб-страница — документ или информационный ресурс Всемирной паутины, доступ к которому осуществляется с помощью веб-браузера.
Сайт, или веб-сайт, также веб-узел, — одна или несколько логически связанных между собой веб-страниц; также место расположения контента сервера.

Основы разметки веб-страницы. Основные понятия и представления о макете веб-страницы.

Интернет - всемирная компьютерная сеть, множество узлов которой составляют взаимодействующие по единым правилам компьютеры и компьютерные устройства.
Интранет – это частная компьютерная сеть, в которой используются стандарты и протоколы Интернета, что значительно повышает эффективность сотрудничества и взаимодействия между работниками организации.
Экстранет – это распределенная информационная среда, объединяющая все филиалы компании, ее партнеров и клиентов.

Языки разметки.

HTML расшифровывается как HyperText Markup Language (язык гипертекстовой разметки). Это язык разметки документов во Всемирной паутине (World Wide Web, WWW).
CSS (Cascading Style Sheets) — язык таблиц стилей, который позволяет прикреплять стиль (например, шрифты и цвет) к структурированным документам (например, документам HTML и приложениям XML).
Язык разметки Java Speech API (JSML) - это язык разметки на основе HML для аннотирования вводимого текста в синтезаторы речи.
