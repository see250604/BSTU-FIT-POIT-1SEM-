HTML5 — язык для структурирования и представления содержимого всемирной паутины. Это пятая версия HTML.
Новые API В дополнение к определению разметки HTML5 устанавливает API, который может быть использован с JavaScript. Возможности DOM расширены и фактически используемые свойства задокументированы.
CSS3 — активно разрабатываемая спецификация CSS. Главной особенностью CSS3 является возможность создавать анимированные элементы без использования JS, поддержка линейных и радиальных градиентов, теней, сглаживания и прочее.
Преимущественно используется как средство описания и оформления внешнего вида веб-страниц
Нововведения, начиная с малых, закруглённых углов блоков, заканчивая трансформацией и, возможно, введением переменных.
